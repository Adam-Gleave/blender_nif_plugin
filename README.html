Blender NIF Scripts v2.1.19: Release Notes

If you find bugs, please report them on the sourceforge tracker, along with a .blend or .nif that caused the error and a brief description of how to reproduce the error. Although we have done our best to squash all bugs we could find, it is unlikely that we have found them all.

Highlights of this Release

  • Apply mode is imported and exported via Blender's texture blending mode. Blending mode MIX corresponds to APPLY_MODULATE, LIGHTEN to APPLY_HILIGHT, and MULTIPLY to APPLY_HILIGHT2.

Using the Scripts

The quick-start guide describes how to prepare your textures and how you can use the scripts. The guide also lists the limitations of the scripts, known issues, and workarounds for some issues.

Installation

Requirements

  • Blender 2.45 or higher.
  • Python 2.5 or higher.
  • PyFFI 0.5.2 or higher.

Manual Install

  • If you installed Blender using the .exe installer, you're best off using the Blender NIF Scripts .exe installer. If you installed Blender from a .zip, then you should download the .zip, extract its contents, and copy the extracted scripts directory to Blender's scripts directory. This is by default C:\Program Files\Blender Foundation\Blender\.blender\scripts. Restart Blender, and the options to import and export NIF files will appear in the File-Import and File-Export menu's. Make sure you have Python and PyFFI installed as well!
  • For use with linux, the .zip includes an install.sh script which copies the relevant files to your ~/.blender folder.
